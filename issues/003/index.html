PAGE 2 — THE TURN

Waiting is not a pause.

Waiting is a place.

You can live there
without calling it temporary.


PAGE 3 — OFFERINGS

a wolf who waits
not for permission
but for alignment

a door left open
so the room can breathe

a map with no destination
and no apology

an arrow that points down
toward the present

a list titled “later”
with nothing under it

a page that refuses to be filled
and wins


PAGE 4 — EXIT

Still counts.
leave this.
